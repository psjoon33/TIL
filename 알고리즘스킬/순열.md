### 순열

```python
arr = [1, 2, 3, 4]
N = len(arr)
visited = [0]*N
sel = [0]*N

def perm(idx):
    if idx == N:
        print(sel)
        
    else:
        for i in range(N):
	        if visited[i] != 1:
                visited[i] = 1
                sel[i] = arr[idx]
                perm(idx+1)
                visited[i] = 0

perm(0)

```
